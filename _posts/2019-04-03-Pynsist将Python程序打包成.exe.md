# Pynsist制作Python程序的Windows安装包

```
使用Python语言编写GUI程序时，往往需要将程序中的依赖包连同程序一同打包成.exe安装文件，
打包完成后，便于在其他Window电脑上安装并使用该程序。本篇文章主要讲解如何使用Pynsist进行打包。
```

## 1. 安装Pnsist
使用conda创建虚拟环境xxx,完成后进入虚拟环境使用pip安装
```
pip install pynsist
```
